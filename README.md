Project Purpose:
This project is a full-stack bookmark application that allows users to efficiently save, view, and manage their favorite web links. It emphasizes a clean, intuitive UI and persistent backend storage to provide a seamless user experience.

Core Technologies & Features:

Frontend: React (with TypeScript), Tailwind CSS, shadcn/UI – Built with functional components for modularity and reusability.

Backend: Node.js, Express.js – Provides RESTful APIs for creating, reading, deleting, and liking bookmarks.

PostgreSQL: Stores bookmarks and user data persistently.

Authentication: All endpoints are protected to ensure secure access.

Link Preview API: Integrates with www.linkpreview.net to fetch metadata (title, description, image) for submitted URLs.

Responsive Design: Mobile-friendly and scalable layout using Tailwind CSS.

Infrastructure & Deployment: Deployed on Vercel for automatic updates on code push.

TypeScript: Enforces static typing for more reliable and maintainable code.

UI/UX Design: Developed with a clean and user-centered approach, inspired by wireframes for consistency and ease of use.

## Getting Started

### Prerequisites

- Node.js (version 2.3.0 or higher)  
- npm (or yarn)

### Installation

1.  Clone the repository:
            `git clone (https://github.com/paulinahalpern/bookmark-app.git)`
    
2.  Navigate to the project directory:
            `cd client` `cd bookmark-app-api`
    
4.  Install dependencies:
            `npm install`
    
5. Run the development server
            `npm run dev`
   
Open `http://localhost:5000` to view the app in your browser.
